CSV Field Guide
📌 REQUIRED FIELDS
  • student_first_name — First name (max 50 chars)
  • student_last_name — Last name (max 50 chars)
  • student_class — Class (e.g. Class 10, STD 11)
  • student_email — Valid email address
✓ OPTIONAL FIELDS
  • student_dob — Date of birth (YYYY-MM-DD)
  • parent_name — Parent/guardian name
  • parent_email — Parent email
  • parent_phone — Parent phone (10-15 digits)
  • city — City name (max 50 chars)
  • notes — Internal notes (max 200 chars)
⚠️ VALIDATION RULES
  • No duplicate email addresses in same upload
  • DOB must result in reasonable student age (8+ years)
  • Class must be a recognized value
  • Emails must be valid format
  • Max 5,000 rows per upload
